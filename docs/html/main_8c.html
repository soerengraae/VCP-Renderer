Detailed Description

VCP Renderer main application entry point.

Initializes all subsystems for the Volume Control Profile Renderer demonstration and starts the Bluetooth advertising process. The application runs event-driven with no main loop to conserve power.

uint8_t init ( )

Initialize all application subsystems.

Initializes peripherals (LED, button) and Bluetooth stack in sequence. Critical failures in LED or Bluetooth will cause initialization to fail. Button failure is non-critical and only generates a warning.

Main application entry point.

Initializes all subsystems and starts the VCP Renderer service. The application is event-driven and does not run a main loop to save power. All functionality is handled through Bluetooth callbacks and work queues.
